Controller Data API

GET all controllers

Returns list of all controllers data

Usage

MethodGET
URL/controllers

Response message body:

AttributeDescriptionExample
idController id1
descDescription of controllerController A
ip_addrIP address of controller192.168.0.101
ctrn_typeController typeRyu 1.3

Example of use:

$ curl -X  http://example.com/controllers

Example result:

[
  {
    'id': 1, 
    'desc': 'Controller In Room 234',
    'ip_addr': '192.168.0.101',
    'type': 'ryu 1.3'
  },
  {

    'id': 2, 
    'desc': 'Controller In Room 235',
    'ip_addr': '192.168.0.102',
    'ctrn_type': 'Open Daylight 1.0'

Switch Data API

Get all switch data from a local controller

Return all switch data from specific controller

Usage

MethodGET
URL/controller/<controller id>/switches

Response message body:

AttributeDescriptionExample

lc_idLocal controller id of switch234
dpidDatapath id of switch00000001
