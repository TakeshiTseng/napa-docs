Controller Data API

GET all controllers

Returns list of all controllers data

Usage

MethodGET
URL/controllers

Response message body:

AttributeDescriptionExample
idController id1
descDescription of controllerController A
ip_addrIP address of controller192.168.0.101
typeController typeRyu 1.3

Example of use:

$ curl -X  http://example.com/controllers

Example result:

[
  {
    'id': 1, 
    'desc': 'Controller In Room 234',
    'ip_addr': '192.168.0.101',
    'type': 'ryu 1.3'
  },
  {

    'id': 2, 
    'desc': 'Controller In Room 235',
    'ip_addr': '192.168.0.102',
    'type': 'Open Daylight 1.0'
  }
]

Get single Controller data

Returns a controller data

Usage

MethodGET
URL/controller/<id>

Response message body:

AttributeDescriptionExample
idController id1
descDescription of controllerController A
ip_addrIP address of controller192.168.0.101
typeController typeRyu 1.3

Example of use:

$ curl –X GET http://example.com/controller/1

Example result:

{
  'id': 1, 
  'desc': 'Controller In Room 234',
  'ip_addr': '192.168.0.101',
  'type': 'ryu 1.3'
}

Insert controller data

Add a controller data into database

Usage

MethodPOST
URL/controller

Request message body:

AttributeDescriptionExample
descDescription of controllerController A
ip_addrIP address of controller192.168.0.101
typeController typeRyu 1.3

Response message body:

AttributeDescriptionExample
idController id1
descDescription of controllerController A
ip_addrIP address of controller192.168.0.101
typeController typeRyu 1.3

Example of use:

$ curl –X POST -d ’{ \
    “desc”: “Controller A”, \
    “ip_addr”: “192.168.0.101” \
    “rype”: “ryu 1.3” \
}’ \
http://example.com/controller

Example result:

{
  'id': 1, 
  'desc': 'Controller In Room 234',
  'ip_addr': '192.168.0.101',
  'type': 'ryu 1.3'
}

Update controller

Update controller data

Usage

MethodPOST
URL/controller/<id>

Request message body:

AttributeDescriptionExample
desc (optional)New escription of controllerController B
ip_addr (optional)New IP address of controller192.168.0.102
type (optional)New Controller typeRyu 1.4

Response message body:

AttributeDescriptionExample
idController id1
descNew escription of controllerController B
ip_addrNew IP address of controller192.168.0.102
typeNew Controller typeRyu 1.4

Example of use:

$ curl –X POST -d ’{ \
    “desc”: “Controller B”, \
    “ip_addr”: “192.168.0.102” \
    “rype”: “ryu 1.4” \
}’ \
http://example.com/controller/1

Example result:

{
  'id': 1, 
  'desc': 'Controller B',
  'ip_addr': '192.168.0.102',
  'type': 'ryu 1.4'
}

Delete Controller Data

Remove Controller data from database

Usage

MethodGET
URL/controller/<id>

Response message body:

AttributeDescriptionExample
idController id1
descNew escription of controllerController B
ip_addrNew IP address of controller192.168.0.102
typeNew Controller typeRyu 1.4

Example of use:

$ curl -X DELETE http://example.com/controller/1

Example result:

{
  'id': 1, 
  'desc': 'Controller B',
  'ip_addr': '192.168.0.102',
  'type': 'ryu 1.4'
}
